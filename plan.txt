План разработки мода "Мурим" для Minecraft с использованием Forge:

1. Настройка проекта:
   a. Создать основную структуру проекта
   b. Настроить build.gradle
   c. настроить главный класс мода (MurimMod.java)
   d. Настроить mcmod.info

2. Система культивации:
   a. Создать enum для уровней культивации
   b. Реализовать класс для хранения данных культивации игрока
   c. Добавить события для обновления и сохранения данных культивации
  1 сначала игрок должен создать дань тань первая культивация будет создавать его последующие заполнять или расширять (проходить. на новые этапы/уровни)
    -нажимая на кнопку ты (садишся) экран затемняется и видно только контур тела и набор клавишь которые нужно нажать за время и с каждым этапом быстрее, и потихонь должна идти анимация как энергия окруджающей среды переходит в низ живота образуя дань тань ядро ци
    если что то не получается анимация прерывается и игрок получает урон 
    - В случае успеха уровень ци/культивации игрока повышается. Если игрок ошибется, медитация провалится.
    - Монстры могут прервать медитацию, если нападут на игрока
    - с каждым уровнем культивации будет сложнее проходить медитацию и становится всё сложнее
    - за каждый уровень культивации будет даваться опыт ци и игрок будет сильнее

3. Базовые предметы:
   a. Создать класс для базового меча мурим
   b. Добавить текстуры и модели для меча
   c. Зарегистрировать меч в игре

4. Система пилюль:

   b. Реализовать эффекты пилюль
   c. Добавить рецепты создания пилюль
   d. Создать GUI для управления пилюлями

   c. Добавить предметы-свитки для изучения техник
   d. Создать GUI для управления изученными техниками
